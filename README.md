# Triple C — Documentation

This folder contains documentation you can use to describe the project (e.g. for a video or onboarding).

| File | Purpose |
|------|--------|
| **MAIN_FLOW_AND_CASHFLOW.md** | Main user and system flow; cash flow (where USDT goes) for all card tiers on mint; CLC caps; first-mint overlap; contract roles. |
| **CARDS_CASHBACK_SPEC.md** | Cards Cashback (referral) spec: conditions 1–4, amounts per tier, 95% referrer / 5% developer, and how referrers qualify. |
| **VIDEO_SCRIPT.md** | Step-by-step script to record a video: intro, connect wallet, loyalty, tiers, mint, CLC, withdraw, cash flow, all cards, contracts summary, outro. |
| **FUNCTION_REFERENCE.md** | List of all contract and frontend functions with short descriptions. |
| **SYSTEM_DIAGRAM.md** | Mermaid diagrams: architecture, cash flow on mint, CLC state, frontend sections, contract roles. |
